## 依赖包的安装

由于依赖了[phantomjs](http://phantomjs.org/)，这个文件下载要翻墙，比较大，所以建议的方式是单独翻墙，然后`npm install phantom phantomjs -g`，装好了phantomjs再npm install其他的依赖。
